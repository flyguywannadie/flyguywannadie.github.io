BALL KICKER

This game was created for the Neumont 2023 Winter Game Jam that lasted 2 days.
I was the lead programmer for the project.


I made the training minigames and rythm game systems.
Before working on this game, I did not know how rythm game systems worked.
My implementation is not perfect on the backend side.
I have done a lot of research after the fact into making the system better.
I have a better understanding of how to integrate a system into a game.


Being my first lead position on a team, I directed code implementations.
I made sure everyone's code was very plug and play, so putting together
my team's code went smoothly.
Once a person had their code done, they put it in a test scene and it would work.
