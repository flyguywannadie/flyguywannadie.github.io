Brad To The Bone

This game was created for the Neumont 2023 Summer Game Jam that lasted 3 days.
I was the character artist in this game, and had a minor role with the programming.



Though my programming contributions were limited, I still had to learn about asset creation.
I made the character visuals along with the animator that controlled Brad himself.
My programming role was to hook up the character controller to the animated Brad.
This taught me the importance to game feel, as I had to adjust my character animations
and add new ones to make the character control well.
