BALL KICKER

This was my final project for my first game development class at Neumont.
It was developed in GameMaker2 over the course of around 2 weeks.


Most of the class was learning GameMaker2, so when It came around to doing the
final project I wanted to do something more.
I learned how to do a title screen, Programmatically add objects to a GameMaker GUI,
Implement velocity and friction to Objects, More advanced Enemy AI, and
create Object Oriented Structure for the different balls.
